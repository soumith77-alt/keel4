# Voice demo audio

Place your Voice AI demo recording here so visitors can listen on the Voice AI service page.

**Required file:** `voice-ai-demo.mp3`

- Path used on the site: `/audio/voice-ai-demo.mp3`
- Supported formats: MP3 is recommended; browsers also support WAV, OGG, M4A.
- To use a different filename, update `demoAudioUrl` in `config/content.ts` for the Voice AI service.
